Ubuntu永久修改DNS

Page content
  • 方法一
  • 方法二
  • 只修改/etc/resolv.conf再重启后会被恢复。

方法一

修改 vi /etc/network/interfaces

# The loopback network interface
auto lo
iface lo inet loopback

# The primary network interface
auto eth0
···
dns-nameservers 114.114.114.114		//增加这一条
···

/etc/init.d/networking restart

方法二

vi /etc/resolvconf/resolv.conf.d/base //文件为空，需要新增

nameserver 8.8.8.8 #希望修改成的DNS
nameserver 114.114.114.114 #希望修改成的DNS

/etc/init.d/networking restart
