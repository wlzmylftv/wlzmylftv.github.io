Node.js在Ubuntu中可用。因此，您所需要做的就是打开一个终端并使用以下命令：

sudo apt install nodejs

要安装npm，请使用以下命令：

sudo apt install npm

我建议也安装npm，因为无论如何您都将需要它。Node.js和npm的大小都不大。

如果要删除Node.js和npm，可以使用以下命令：

sudo apt remove nodejs npm
